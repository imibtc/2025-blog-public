**一、注册中文域名（以阿里云为例）**
--------------------

**1.搜索域名**

 - 进入阿里云域名注册页：https://wanwang.aliyun.com/
 - 搜索框输入`伊米博客`，筛选「中文域名」标签页
 - 推荐后缀：.我爱你（19元/年） > .网络（180元/年） > .游戏（550元/年）

**2.​​实名认证​**

购买后需创建「个人」信息模板并实名认证（审核约20分钟）

![](/blogs/Chinese-name/b8d4103b6ab547c3.webp)

**二、购买服务器空间（免备案）**
------------------

**1.​​选低价香港/美国空间​**

 - 闲鱼/淘宝搜香港免备案空间，选10M容量款（约2元/年）
 - 我用的是：[免备案空间][3]

**2.​​绑定域名​**

 - 在空间面板→「基本配置」→「域名绑定」处添加中文域名
 -「添加域名」界面，粘贴`伊米博客.我爱你`

**三、域名解析设置**
------------

**1.​​配置解析记录​**

 - 回到阿里云控制台 → 域名「解析设置」
 - 添加记录：
　　1，类型：A记录
　　2，主机名：@（主域名）
　　3，记录值：​​粘贴服务器IP​​（从空间面板获取）

**四、部署跳转代码（二选一）**
-----------------

**​​下载代码模板​​：**

https://pan.quark.cn/s/1c052454a10b

-----------------

> ⚠️ 上传前删除默认文件名后缀（保留index.html）

**方案1：无引导页（直接跳转）**

 - 修改index.html： `<meta http-equiv="refresh" content="0;url=https://你的目标链接.com">`
 - ​​适用场景​​：跳转网盘/QQ加群链接等即时操作

**方案2：有引导页（倒计时跳转）**

 - 修改index.html：`window.location.href = 'https://目标链接.com'; // 跳转地址`
 - 适用场景​​：跳转网址网站等操作

**五、上传代码到服务器**
--------------

1，登录空间面板 →「数据管理」→「在线文件管理」
2，删除默认index.html，上传修改后的代码文件
